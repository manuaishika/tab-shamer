# use it before its on the app store

## 1. Install
```bash
pnpm install
```

## 2. Create Icon
- Open `scripts/create-icon.html` in browser
- Click download button
- Save as `public/icon.png`

## 3. Build & Load
```bash
pnpm build
```

Then:
- Open Chrome → `chrome://extensions/`
- Turn on "Developer mode"
- Click "Load unpacked"
- Select the `dist` folder

## DONE! 🎉

Open 20+ tabs to see it work!
